astro-koharu ブログテーマへようこそ！

## このテーマについて

astro-koharu は Astro 5.x をベースにしたモダンなブログシステムで、Hexo の Shoka テーマにインスピレーションを受けています。主な特徴：

- **高パフォーマンス** - Astro による静的サイト生成で、超高速な読み込みを実現
- **エレガントなデザイン** - アニメ風の美学とピンク＆ブルーの配色
- **豊富な機能** - マルチレベルカテゴリー、タグ、目次、検索など
- **レスポンシブ** - デスクトップとモバイルに完璧に対応

## クイックスタート

### 1. ブログを設定

`config/site.yaml` ファイルを編集：

```yaml
site:
  title: あなたのブログ名
  author: あなたの名前
  description: ブログの説明
  # ...その他のオプション
```

### 2. 最初の投稿を書く

`src/content/blog/` ディレクトリに Markdown ファイルを作成：

```markdown
---
title: 初めての投稿
date: 2024-01-01
tags:
  - タグ1
categories:
  - カテゴリー名
---

投稿の内容...
```

### 3. デプロイ

Vercel で簡単にデプロイ：
