この投稿では、目次の自動番号を無効にする方法を紹介します。

## 目次番号機能

デフォルトでは、astro-koharu は CSS カウンターを使って目次に階層的な番号を自動付与します：

- 1. 第一章
  - 1.1. セクション 1
  - 1.2. セクション 2
- 2. 第二章

## 番号を無効にする

特定の投稿で番号を無効にするには、`tocNumbering: false` を設定します：

```yaml
---
title: 私の投稿
tocNumbering: false
---
```

## 比較

### 番号あり（デフォルト）

目次の項目に 1.、1.1.、1.1.1. のような番号が表示されます。

### 番号なし

目次の項目は見出しテキストのみで、番号プレフィックスはありません。

## この投稿の効果

この投稿では `tocNumbering: false` が設定されています。サイドバーの目次（デスクトップ）を確認するか、目次を展開（モバイル）して結果をご覧ください。

## 技術的な実装

番号付けは純粋な CSS カウンターで実装されており、ランタイムオーバーヘッドはゼロです：

```css
.toc-numbering {
  counter-reset: h2;
}

.toc-numbering h2::before {
  counter-increment: h2;
  content: counter(h2) ". ";
}
```

## 無効にするタイミング

以下のような場合に番号を無効にすることを検討してください：

- エッセイ風の投稿
- すでに番号がある見出し
- 構造がゆるい投稿
- 個人的な好み

## まとめ

目次の番号付けはオプション機能です — コンテンツの種類に合わせて柔軟に使い分けましょう。
